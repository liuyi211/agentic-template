# Examples

This directory contains example projects demonstrating how to use the Agentic Template in real scenarios.

## Available Examples

### [react-dashboard](react-dashboard/)

A React + TypeScript + Ant Design admin dashboard.

**Tech Stack**: React 18 + Vite + TypeScript + Ant Design 5 + Zustand

**Current Stage**: Stage 01 - Authentication Module (60%)

### [node-api](node-api/)

A Node.js + Fastify + Prisma REST API service.

**Tech Stack**: Node.js 20 + Fastify 4 + Prisma 5 + PostgreSQL 15 + Zod

**Current Stage**: Stage 01 - Core Infrastructure (80%)

### [nextjs-fullstack](nextjs-fullstack/)

A Next.js 14 fullstack blog platform.

**Tech Stack**: Next.js 14 + React 18 + TypeScript + Prisma + PostgreSQL + Tailwind CSS + shadcn/ui

**Current Stage**: Stage 00 - Project Initialization (40%)

## How to Use Examples

1. Copy an example directory to your project
2. Rename and modify `AGENTS.md` with your project info
3. Fill in all `memory-bank/` templates with your actual project details
4. Start collaborating with your AI assistant

## Note

These examples are templates - they don't contain actual runnable code. They demonstrate:
- How to structure your Agentic Template project
- How to fill in the design documents
- How to track progress with todo.md and progress/
- How to define tech-stack constraints
